# Psyche, You Thought!
This is a browser-based educational party game inspired by NASA’s Psyche mission, where players compete against an NPC opponent by selecting creative response cards, completing mini-games, and progressing through a space-themed adventure toward the Psyche asteroid. Built using Unity and C#, the game combines card-based strategy, mission progression, and six interactive mini-games—including Flappy Bird, Matching Cards, Whac-A-Mole, Space Invaders, a Rhythm Game, and Dice Battles—to create an engaging and educational gameplay experience.
